# 2026 Changelog

### ClickHouse release v26.3.8.4-lts (f3c6e5a4d27) FIXME as compared to v26.3.7.8-lts (177c6aa7da1)

#### Bug Fix (user-visible misbehavior in an official stable release)
* Backported in [#102453](https://github.com/ClickHouse/ClickHouse/issues/102453): Coalescing merge tree fix for array type. This closes [#89509](https://github.com/ClickHouse/ClickHouse/issues/89509). [#102384](https://github.com/ClickHouse/ClickHouse/pull/102384) ([Konstantin Vedernikov](https://github.com/scanhex12)).
* Backported in [#102479](https://github.com/ClickHouse/ClickHouse/issues/102479): Fix out-of-bounds read in string search functions (`countSubstrings`, `position`, etc.) when searching for a needle consisting entirely of null bytes. [#102401](https://github.com/ClickHouse/ClickHouse/pull/102401) ([Raúl Marín](https://github.com/Algunenano)).
* Backported in [#102456](https://github.com/ClickHouse/ClickHouse/issues/102456): Full text index settings (`enable_full_text_index`, `allow_experimental_full_text_index`, `use_skip_indexes_on_data_read`) are no longer disabled when the `compatibility` setting points to a version older than 26.1. Previously this could prevent `SharedDatabaseCatalog` from creating tables with text indexes. [#102422](https://github.com/ClickHouse/ClickHouse/pull/102422) ([Nikita Fomichev](https://github.com/fm4v)).

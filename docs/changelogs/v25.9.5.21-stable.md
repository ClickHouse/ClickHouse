# 2025 Changelog

### ClickHouse release v25.9.5.21-stable (57cfa8fbbf8) FIXME as compared to v25.9.4.58-stable (08afb4f28ee)

#### Bug Fix (user-visible misbehavior in an official stable release)
* Backported in [#88511](https://github.com/ClickHouse/ClickHouse/issues/88511): Fix reading null map subcolumn from Variants that cannot be inside Nullable. [#87798](https://github.com/ClickHouse/ClickHouse/pull/87798) ([Pavel Kruglov](https://github.com/Avogar)).
* Backported in [#88734](https://github.com/ClickHouse/ClickHouse/issues/88734): A `Poco::TimeoutException` exception thrown from `Poco::Net::HTTPChunkedStreamBuf::readFromDevice` leads to a crash with SIGABRT. [#88668](https://github.com/ClickHouse/ClickHouse/pull/88668) ([Mi—Åhael Stetsyuk](https://github.com/mstetsyuk)).
* Backported in [#88908](https://github.com/ClickHouse/ClickHouse/issues/88908): After recovering, a Replicated database replica might get stuck for a long time printing messages like `Failed to marked query-0004647339 as finished (finished=No node, synced=No node)`, it's fixed. [#88671](https://github.com/ClickHouse/ClickHouse/pull/88671) ([Alexander Tokmakov](https://github.com/tavplubix)).
* Backported in [#88892](https://github.com/ClickHouse/ClickHouse/issues/88892): Fixes access validation on select for `loop` table function. [#88802](https://github.com/ClickHouse/ClickHouse/pull/88802) ([pufit](https://github.com/pufit)).
* Backported in [#88942](https://github.com/ClickHouse/ClickHouse/issues/88942): Fix bug in the function `reverseUTF8`. In previous versions, it mistakenly reversed the bytes of UTF-8 code points of length 4. This closes [#88913](https://github.com/ClickHouse/ClickHouse/issues/88913). [#88914](https://github.com/ClickHouse/ClickHouse/pull/88914) ([Alexey Milovidov](https://github.com/alexey-milovidov)).
* Backported in [#88978](https://github.com/ClickHouse/ClickHouse/issues/88978): Do not check access `SET DEFINER <current_user>:definer` when creating a view with SQL SECURITY DEFINER. [#88968](https://github.com/ClickHouse/ClickHouse/pull/88968) ([pufit](https://github.com/pufit)).
* Backported in [#89083](https://github.com/ClickHouse/ClickHouse/issues/89083): This closes [#88081](https://github.com/ClickHouse/ClickHouse/issues/88081). [#88988](https://github.com/ClickHouse/ClickHouse/pull/88988) ([scanhex12](https://github.com/scanhex12)).
* Backported in [#89026](https://github.com/ClickHouse/ClickHouse/issues/89026): Fixed performance degradation in analysis of skipping indexes. [#89004](https://github.com/ClickHouse/ClickHouse/pull/89004) ([Anton Popov](https://github.com/CurtizJ)).

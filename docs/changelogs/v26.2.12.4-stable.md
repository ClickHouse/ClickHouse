# 2026 Changelog

### ClickHouse release v26.2.12.4-stable (8aacd0fc6dd) FIXME as compared to v26.2.11.5-stable (f6f848d87c4)

#### Bug Fix (user-visible misbehavior in an official stable release)
* Backported in [#102451](https://github.com/ClickHouse/ClickHouse/issues/102451): Coalescing merge tree fix for array type. This closes [#89509](https://github.com/ClickHouse/ClickHouse/issues/89509). [#102384](https://github.com/ClickHouse/ClickHouse/pull/102384) ([Konstantin Vedernikov](https://github.com/scanhex12)).
* Backported in [#102477](https://github.com/ClickHouse/ClickHouse/issues/102477): Fix out-of-bounds read in string search functions (`countSubstrings`, `position`, etc.) when searching for a needle consisting entirely of null bytes. [#102401](https://github.com/ClickHouse/ClickHouse/pull/102401) ([Raúl Marín](https://github.com/Algunenano)).
* Backported in [#102469](https://github.com/ClickHouse/ClickHouse/issues/102469): Full text index settings (`enable_full_text_index`, `allow_experimental_full_text_index`, `use_skip_indexes_on_data_read`) are no longer disabled when the `compatibility` setting points to a version older than 26.1. Previously this could prevent `SharedDatabaseCatalog` from creating tables with text indexes. [#102422](https://github.com/ClickHouse/ClickHouse/pull/102422) ([Nikita Fomichev](https://github.com/fm4v)).

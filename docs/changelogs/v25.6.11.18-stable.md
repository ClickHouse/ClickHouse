Closes [#86122](https://github.com/ClickHouse/ClickHouse/issues/86122). Closes https://github.com/ClickHouse/clickhouse-core-incidents/issues/473. The range specified is invalid for the current size of the resource` when reading empty blobs from Azure blob storage for plain_rewritable disk.

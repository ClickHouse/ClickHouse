# 2026 Changelog

### ClickHouse release v26.3.6.5-lts (76a4a77bb24) FIXME as compared to v26.3.5.12-lts (172e4255c5c)

#### New Feature
* Backported in [#102253](https://github.com/ClickHouse/ClickHouse/issues/102253): Add asynchronous metrics for kernel TCP receive and transmit buffer memory (`sk_rmem_alloc`, `sk_wmem_alloc`) of HTTP connection pool sockets, reported as p50/p75/p90/p95 percentiles and totals per connection group. [#100575](https://github.com/ClickHouse/ClickHouse/pull/100575) ([Sema Checherinda](https://github.com/CheSema)).

#### Bug Fix (user-visible misbehavior in an official stable release)
* Backported in [#102296](https://github.com/ClickHouse/ClickHouse/issues/102296): Fix possible logical error during reading Map subcolumns. Closes [#100769](https://github.com/ClickHouse/ClickHouse/issues/100769). Closes [#101336](https://github.com/ClickHouse/ClickHouse/issues/101336). [#101641](https://github.com/ClickHouse/ClickHouse/pull/101641) ([Pavel Kruglov](https://github.com/Avogar)).
* Backported in [#102267](https://github.com/ClickHouse/ClickHouse/issues/102267): Fix SLRU race bug in filesystem cache 26.1+, which can lead to space reservation logical error. In debug build it can also lead to failed assert: `'Previous state is Evicting, but expected state to be Active while setting Evicting flag for 2c1e3484ecdc6b78a8978fa5b17c5097:0:339 (state: Evicting)'.`. [#101991](https://github.com/ClickHouse/ClickHouse/pull/101991) ([Kseniia Sumarokova](https://github.com/kssenii)).
* Backported in [#102174](https://github.com/ClickHouse/ClickHouse/issues/102174): Fix crash in UDF refresh caused by `ZooKeeperRetriesControl` retrying on a stale (expired) ZooKeeper session without renewing it. [#102059](https://github.com/ClickHouse/ClickHouse/pull/102059) ([Nikita Fomichev](https://github.com/fm4v)).

# 2025 Changelog

### ClickHouse release v25.6.10.16-stable (6007a7b814a) FIXME as compared to v25.6.9.98-stable (b242c4182b8)

#### Improvement
* Backported in [#86824](https://github.com/ClickHouse/ClickHouse/issues/86824): Improve performance of RemoveRecursive request in Keeper. [#86789](https://github.com/ClickHouse/ClickHouse/pull/86789) ([Antonio Andelic](https://github.com/antonio2368)).

#### Bug Fix (user-visible misbehavior in an official stable release)
* Backported in [#86610](https://github.com/ClickHouse/ClickHouse/issues/86610): Shutdown tables properly when recovering database replica. Improper shutdown would lead to LOGICAL_ERROR for some table engines during database replica recovery. [#84744](https://github.com/ClickHouse/ClickHouse/pull/84744) ([Antonio Andelic](https://github.com/antonio2368)).
* Backported in [#86452](https://github.com/ClickHouse/ClickHouse/issues/86452): Fixed a bug in Materialized Views: an MV might not work if it was created, dropped, and then created again with the same name. [#86413](https://github.com/ClickHouse/ClickHouse/pull/86413) ([Alexander Tokmakov](https://github.com/tavplubix)).
* Backported in [#86619](https://github.com/ClickHouse/ClickHouse/issues/86619): Ignore exceptions during flushing log on shutdown and make shutdown more safe (to avoid SIGSEGV). [#86546](https://github.com/ClickHouse/ClickHouse/pull/86546) ([Azat Khuzhin](https://github.com/azat)).

# 2026 Changelog

### ClickHouse release v25.10.7.6-stable (16b240cb400) FIXME as compared to v25.10.6.36-stable (853dbd2ee0c)

#### Bug Fix (user-visible misbehavior in an official stable release)
* Backported in [#103748](https://github.com/ClickHouse/ClickHouse/issues/103748): Plain INSERTs without materialized views no longer request excessive ConcurrencyControl slots and threads (`max_threads` instead of `max_insert_threads`), preventing CC slot starvation and thread count blowup on clusters with high INSERT throughput. [#102961](https://github.com/ClickHouse/ClickHouse/pull/102961) ([Sema Checherinda](https://github.com/CheSema)).

#### Build/Testing/Packaging Improvement
* Upgrade distroless Docker base image from Debian 12 to Debian 13 to eliminate CVEs in system libraries. [#103562](https://github.com/ClickHouse/ClickHouse/pull/103562) ([Rahul Nair](https://github.com/motsc)).
* Refresh distroless Docker base image to fix OpenSSL CVEs in `libssl3t64`. [#103595](https://github.com/ClickHouse/ClickHouse/pull/103595) ([Rahul Nair](https://github.com/motsc)).

#### NOT FOR CHANGELOG / INSIGNIFICANT

* Accept `COMMENT` before `AS SELECT` in view parser for forward compatibility with newer versions. [#99559](https://github.com/ClickHouse/ClickHouse/pull/99559) ([Alexey Milovidov](https://github.com/alexey-milovidov)).

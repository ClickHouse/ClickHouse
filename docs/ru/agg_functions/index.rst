Агрегатные функции
==================

Агрегатные функции работают в `привычном <http://www.sql-tutorial.com/sql-aggregate-functions-sql-tutorial>`_ для специалистов по базам данных смысле.

ClickHouse поддерживает также:

  * :ref:`Параметрические агрегатные функции <aggregate_functions_parametric>`, которые помимо стоблцов принимаю и другие параметры.
  * :ref:`Комбинаторы <aggregate_functions_combinators>`, которые изменяют поведение агрегатных фунций.

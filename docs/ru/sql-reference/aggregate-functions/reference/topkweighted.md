# topKWeighted {#topkweighted}

Аналогична [topK](topk.md), но дополнительно принимает положительный целочисленный параметр `weight`. Каждое значение учитывается `weight` раз при расчёте частоты.

**Синтаксис**

``` sql
topKWeighted(N)(x, weight)
```

**Аргументы**

-   `N` — количество элементов для выдачи.
-   `x` — значение.
-   `weight` — вес. [UInt64](../../../sql-reference/data-types/int-uint.md).

**Возвращаемое значение**

Возвращает массив значений с максимально приближенной суммой весов.

**Пример**

Запрос:

``` sql
SELECT topKWeighted(10)(number, number) FROM numbers(1000)
```

Результат:

``` text
┌─topKWeighted(10)(number, number)──────────┐
│ [999,998,997,996,995,994,993,992,991,990] │
└───────────────────────────────────────────┘
```

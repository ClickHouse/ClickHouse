# 数据库引擎 {#shu-ju-ku-yin-qing}

您使用的所有表都是由数据库引擎所提供的

默认情况下，ClickHouse使用自己的数据库引擎，该引擎提供可配置的[表引擎](../../engines/database-engines/index.md)和[所有支持的SQL语法](../../engines/database-engines/index.md).

除此之外，您还可以选择使用以下的数据库引擎：

-   [MySQL](mysql.md)

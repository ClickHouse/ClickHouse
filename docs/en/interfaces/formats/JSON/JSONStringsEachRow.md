## Description

Differs from JSONEachRow only in that data fields are output in strings, not in typed JSON values.

## Example Usage

Example:

```json
{"num":"42","str":"hello","arr":"[0,1]"}
{"num":"43","str":"hello","arr":"[0,1,2]"}
{"num":"44","str":"hello","arr":"[0,1,2,3]"}
```

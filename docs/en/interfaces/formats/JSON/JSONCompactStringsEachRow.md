## Description

Differs from JSONCompactEachRow only in that data fields are output in strings, not in typed JSON values.

## Example Usage

Example:

```json
["42", "hello", "[0,1]"]
["43", "hello", "[0,1,2]"]
["44", "hello", "[0,1,2,3]"]
```

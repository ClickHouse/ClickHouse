## Description {#description}

This table does not exist if this node is not configured to run an in-process ClickHouse Keeper. It contains one row per Raft cluster member, fusing static cluster topology (from the Raft configuration) with the local node's own log position.

Every node fills exactly one `last_log_index` value — the row matching its own `server_id`. Peer log positions are not surfaced here because they are tracked only on the leader and that view is not symmetric across the cluster.

## Columns {#columns}

<!--AUTOGENERATED_START-->
-   `server_id` ([Int32](../../sql-reference/data-types/int-uint.md)) — Raft server id of this cluster member.
-   `host` ([String](../../sql-reference/data-types/string.md)) — Host parsed from the endpoint (prefix before the last colon).
-   `endpoint` ([String](../../sql-reference/data-types/string.md)) — Raw Raft endpoint as configured (`host:port`).
-   `is_observer` ([Bool](../../sql-reference/data-types/boolean.md)) — `true` if this member is a non-voting observer, configured via `<can_become_leader>false</can_become_leader>`.
-   `priority` ([Int32](../../sql-reference/data-types/int-uint.md)) — Raft priority of this member; higher values are preferred during leader election.
-   `is_leader` ([Bool](../../sql-reference/data-types/boolean.md)) — `true` if this member is the current Raft leader.
-   `is_self` ([Bool](../../sql-reference/data-types/boolean.md)) — `true` if this row describes the local Keeper node.
-   `last_log_index` ([Nullable(UInt64)](../../sql-reference/data-types/int-uint.md)) — Highest log index in the local Raft log store on this node. Populated only on the row matching the current node (`is_self = true`); `NULL` for other rows.
<!--AUTOGENERATED_END-->

```sql
SELECT * FROM system.keeper_cluster ORDER BY server_id;
```

## Description {#description}

This table does not exist if this node is not configured to run an in-process ClickHouse Keeper. It contains one row per Raft changelog file (`changelog_<from>_<to>.bin[.zstd]`) tracked by the in-process Keeper log store, including the active file currently being appended to.

## Columns {#columns}

<!--AUTOGENERATED_START-->
-   `from_log_index` ([UInt64](../../sql-reference/data-types/int-uint.md)) тАФ First Raft log index in the file (inclusive).
-   `to_log_index` ([UInt64](../../sql-reference/data-types/int-uint.md)) тАФ Last Raft log index covered by the filename (inclusive). For the active file this is the rotation target and may be ahead of `last_entry_index`.
-   `last_entry_index` ([Nullable(UInt64)](../../sql-reference/data-types/int-uint.md)) тАФ Highest log index actually appended to this file. `NULL` if the active file has not received any entries yet or the file is broken.
-   `entries` ([UInt64](../../sql-reference/data-types/int-uint.md)) тАФ Number of entries appended to this file. `ALIAS` for `ifNull(last_entry_index - from_log_index + 1, 0)`.
-   `path` ([String](../../sql-reference/data-types/string.md)) тАФ File path on the disk.
-   `disk_name` ([String](../../sql-reference/data-types/string.md)) тАФ Name of the disk holding the file.
-   `size_bytes` ([UInt64](../../sql-reference/data-types/int-uint.md)) тАФ Size of the file on disk.
-   `modification_time` ([DateTime](../../sql-reference/data-types/datetime.md)) тАФ Last modification time of the file.
-   `is_compressed` ([Bool](../../sql-reference/data-types/boolean.md)) тАФ File payload is zstd-compressed.
-   `active` ([Bool](../../sql-reference/data-types/boolean.md)) тАФ This file is currently being appended to.
-   `is_broken` ([Bool](../../sql-reference/data-types/boolean.md)) тАФ Trailing record was found corrupted at startup.
<!--AUTOGENERATED_END-->

Example:

```sql
SELECT from_log_index, to_log_index, entries, path, active FROM system.keeper_changelogs ORDER BY from_log_index;
```

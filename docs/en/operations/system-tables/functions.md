# functions

Contains information about normal and aggregate functions.

Columns:

-   `name`(`String`) – The name of the function.
-   `is_aggregate`(`UInt8`) — Whether the function is aggregate.

**Example**

```sql
 SELECT * FROM system.functions LIMIT 10;
```

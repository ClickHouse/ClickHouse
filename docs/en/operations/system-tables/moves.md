## Description {#description}

The table contains information about in-progress [data part moves](/sql-reference/statements/alter/partition#move-partitionpart) of [MergeTree](/engines/table-engines/mergetree-family/mergetree.md) tables. Each data part movement is represented by a single row.

## Columns {#columns}

<!--AUTOGENERATED_START-->
- `database` ([String](/sql-reference/data-types/string)) вҖ” Name of the database.
- `table` ([String](/sql-reference/data-types/string)) вҖ” Name of the table containing moving data part.
- `elapsed` ([Float64](/sql-reference/data-types/float)) вҖ” Time elapsed (in seconds) since data part movement started.
- `target_disk_name` ([String](/sql-reference/data-types/string)) вҖ” Name of disk to which the data part is moving.
- `target_disk_path` ([String](/sql-reference/data-types/string)) вҖ” Path to the mount point of the disk in the file system.
- `part_name` ([String](/sql-reference/data-types/string)) вҖ” Name of the data part being moved.
- `part_size` ([UInt64](/sql-reference/data-types/int-uint)) вҖ” Data part size.
- `thread_id` ([UInt64](/sql-reference/data-types/int-uint)) вҖ” Identifier of a thread performing the movement.
<!--AUTOGENERATED_END-->

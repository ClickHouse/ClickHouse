# 日付 {#date}

デートだ 1970-01-01(符号なし)からの日数として二バイトで格納されます。 unixエポックの開始直後から、コンパイル段階で定数によって定義された上限しきい値に値を格納できます(現在、これは2106年までですが、完全にサポート
最小値は0000-00-00として出力されます。

日付の値は、タイムゾーンなしで格納されます。

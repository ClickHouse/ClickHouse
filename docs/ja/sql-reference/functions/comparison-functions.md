# 比較関数 {#comparison-functions}

比較関数は常に0または1(Uint8)を返します。

次のタイプを比較できます:

-   数字
-   文字列と固定文字列
-   日付
-   日付と時刻

各グループ内ではなく、異なるグループ間。

たとえば、日付と文字列を比較することはできません。 文字列を日付に変換するには関数を使用する必要があります。

文字列はバイト単位で比較されます。 短い文字列は、それで始まり、少なくとも一つ以上の文字を含むすべての文字列よりも小さくなります。

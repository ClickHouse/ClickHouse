# Null {#null}

Nullテーブルに書き込む場合、データは無視されます。 Nullテーブルから読み取る場合、応答は空です。

ただし、Nullテーブルにマテリアライズドビューを作成できます。 したがって、テーブルに書き込まれたデータはビュー内で終了します。

# ヌル {#null}

Nullテーブルに書き込む場合、データは無視されます。 Nullテーブルから読み取る場合、応答は空です。

ただし、nullテーブルにマテリアライズドビューを作成できます。 したがって、テーブルに書き込まれたデータはビューに表示されます。

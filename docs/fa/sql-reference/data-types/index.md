# انواع داده ها {#data_types}

تاتر می توانید انواع مختلف داده ها در سلول های جدول ذخیره کنید.

این بخش انواع داده های پشتیبانی شده و ملاحظات ویژه ای را برای استفاده و/یا در صورت وجود اجرا می کند.
